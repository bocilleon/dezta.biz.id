Cara Membuat Website Gratis di GitHub Pages

Oleh Anda • 25 Maret 2024 • 5 Menit Baca

Memiliki website sendiri sekarang tidak harus mahal. Dengan GitHub Pages, Anda bisa memiliki website profesional secara gratis selamanya.

Mengapa GitHub Pages?

GitHub Pages sangat cocok untuk pemula karena kehandalannya dan kemudahan integrasinya dengan Git. Anda tidak perlu memikirkan biaya server atau pemeliharaan teknis yang rumit.

Tips: Pastikan Anda menggunakan file index.html sebagai halaman utama Anda agar otomatis terbaca oleh server GitHub.

Langkah Pertama

Langkah pertama adalah membuat repository publik di GitHub dengan nama khusus: username.github.io. Ini adalah kunci agar layanan GitHub Pages aktif secara otomatis.

Setelah itu, Anda tinggal mengupload file HTML dan CSS Anda. Dalam hitungan detik, website Anda akan online dan bisa diakses oleh siapa saja.
